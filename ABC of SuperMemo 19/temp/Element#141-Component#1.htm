●SuperMemo使用两种基本元素类型：topics（摘录卡片）和items（问答卡片）。它们在复习过程中受到不同的对待。摘录卡片代表你想知道的，而问答卡片代表你已经知道的。要详细的了解其中差异，请参阅：摘录卡片和问答卡片。

●保持你的摘录卡片和文章在一个可控的比例内。使用你的分类标准，确保你每天除了阅读之外，还有足量的问答卡片用于复习。摘录卡片过多可能会减缓复习问答卡片的进程，并破坏主动召回*过程。摘录卡片占据的比例过大将使SuperMemo变得像效率低下的传统阅读一样。你可以根据复习所需的时间来决定你的最佳比例。例如，问答卡片(5)：摘录卡片(1)可能仍然会让你花更多的时间在阅读上，而不是在复习上。提高这个比率可以提高你的记忆力。如果你需要大量阅读，可以暂时降低这个比率。如果不确定，请将items:topics设置为3:1到8:1之间的任意位置。还是很难决定？那就将未完成队列的复习计划设置为：每复习1个摘录卡片就复习5个问答卡片。

生成完形填空

●在渐进阅读中，你应该努力把被动记忆的文本转化为主动回忆的问题。理想情况下，所有被动记忆的文本最终都应该删除，因为所有来自“外部世界”的干扰都会降低SuperMemo的精确度。被动记忆的文本在学习中几乎不能提供额外的帮助，只能干扰你高效的复习节奏。所以，这些文本应该只用于生成新的完形填空（或供参考）

●按Alt+Z时，可以将当前摘录卡片中选定的关键字挖空为完形填空。新创建的问答卡片不可见（即：您不会立即看到答案或删除括号[...]）。按Alt+←可以看到新创建的问答卡片。使用该键编辑新创建的完形填空（例如添加参考上下文、缩短文本、改进措辞等）。但是，如果可能的话，您应该渐进（循序渐进）地执行这样的小任务，即在下一次遇到该完形填空时再编辑它。

●如果要挖空多个关键字，则在生成完形填空之前，应确保处理的语句或段落尽可能简单。你应该试着只用一个句子来生成完形填空！一些新手起初不喜欢渐进学习。造成他们消极情绪的主要原因就是那些奇奇怪怪的完形填空。将段落简化为一个简单的句子，再将其生成简单的完形填空，只需很少的处理。在复杂文本上使用完形填空会增加在简化文本时重新编辑的成本。如果你将一个较长的多个句子组成的段落生成完形填空，你将不得不付出额外的努力来简化问答卡片。因此，所有的完形填空都应该足够简洁，以确保您在复习的时候可以完全读懂它们。否则，你的大脑会倾向于从非语义线索“推断”出答案。这会使学习目的背道而驰！确保一个句子挖空一个关键字生成完形填空，您可以大大节省复习时间和简化完形填空所需的时间，并根据最小信息原则将其转换为最终形式。

●摘录片段、生成完形填空并编辑它们的工作应该是渐进的。在每次复习中，尽可能多的处理学习材料是很有必要的！在复习之余生成完形填空并且对完形填空进行编辑可以为学习带来额外的好处，而且更省时。每当你回忆知识时，你就会强化记忆中给定知识的表述和“连接性”。此外，随着学习的进行，你的优先级也会发生变化。有时，你会把主要精力放在很快变得无关紧要或过时的知识上面。渐进方法降低将注意力分配不均造成的影响。因此，不仅在阅读时，而且在后续的知识处理和形成过程中，都应采用渐进主义。下面提供了一个示例。不要一次性生成所有的完形填空。应该使其递增，在今天生成一个完形填空，在下次复习时生成另一个完形填空。学习过程的渐进性、在学习过程中使用各种颜色标记文本以及复杂的摘录层次似乎与许多人的完美主义相矛盾。然而，渐进阅读的目的是在最小时间内达到最大效果。因此，在挖空时，您已经对挖空的句子被动的产生了记忆痕迹。因此，最佳策略不是继续生成完形填空，而是转到队列中的其他元素或同一文章中的其他摘录（如果这篇文章是最高优先级）

模拟现实生活中的情景来对抗记忆干扰

有些文本中的知识类型记忆起来非常困难。比如列表和集合形式。即使你对非洲地图了如指掌，让你回答“列出非洲所有国家”的要求可能相当困难。有一些行之有效的技巧可以帮助你使用SuperMemo处理重复的、包含列表的或枯燥的知识。所有在记忆阶段的付出都是高昂的，但从长远来看，由于遗忘率较低，将得到更为丰厚的回报。这其中，2个基本原则是：

1逐渐地将各部分知识“粘到”你的整体知识结构中

2尽可能的直观和便于记忆

以下是一些具体的提示：

●使用思维导图：在网上搜索对应知识点的图片（如非洲政治地图）。这张照片将为你的记忆提供基本的框架。就像圣诞树上的装饰品一样，你会把新知识挂在这个记忆框架上。使用图片说明所研究概念组中的所有摘录卡片和问答卡片。

●不要一下子全部学会：当单个items获得某种特殊意义时，逐渐添加它们。当它们与你的其他知识相吻合时，再加上它们。当您特别需要或了解相关主题时，再添加它们。如果你在考试中需要列举式的知识，用传统的方法来填鸭式的学习，当你觉得这些知识很有趣或者很重要的时候，你还可以在独特的上下文中继续添加它们。

●用故事联想：如果你问一位该领域的专家，你可能会听到他通过与联想故事而掌握了列列表的知识。他或她在学校学到的东西很快就被遗忘了，但要解决的个别案例或问题因其独特性而留下了良好而持久的印记。一旦你发现了一个与你难以记忆的问答卡片相关的故事，试着在这个故事的背景下学习这些问答卡片（例如，只有在读到埃托奥转会切尔西的消息时，才把喀麦隆“挂”在你的知识树上）。如果你在练习过程中联想到了故事，请简短描述，并作为上下文。

●用渐进阅读补充：不要将所有的items都按照相同的重复和单调的模板进行编排，尝试在不同的工作情景中使用渐进式阅读来生成完形填空。理想情况下，你可以在复习topic时生成一个副标题（例如，关于非洲的一个国家）。选择一个似乎最明显或最重要的知识来记忆。试着添加一些补充材料，但是不要添加一些令你无需思考就可以正确回答问题的线索。

●与专家比较：询问该领域的专家他是如何记住给定的事实或关联的。在某些情况下，你可能会沮丧地发现，专家们对大学必修课的记忆是多么糟糕。在其他时候，你会看到他们的记忆是如何通过简单的记忆法轻松解决问题的。这将有助于你在一个压缩的时间尺度上模拟现实生活中的学习，而不会浪费时间去试图掌握别人无论如何也无法掌握的东西。这就是学校学习和你高效的渐进学习之间的基本区别：你不会按照一个严格的学习方案用光所有时间和精力。你将用你的创造力逐步建立一座知识大厦！

示例：处理列举

如果你在学习有关地质时期的知识，你一定会产生抵触情绪，特别是你才接触这个主题的时候。使用自上而下的学习规则，确保你在学习纪之前，能清楚地知道每个代。

一个典型的错误是从意义不明的段落开始学习。例如，古生代层序：“寒武纪、奥陶纪、志留纪、泥盆纪、石炭纪、二叠纪”可能会生成问题：古生代层序：“寒武纪、奥陶纪、[…]、泥盆纪、石炭纪、二叠纪”。这种拎不清的知识点会让初学的人感到不安。换言之，只有那些头脑中已经储存好这些知识的人，才能够以很低的成本解决这类问题！结论：除非你已经知道你要尝试怎么学习，否则学习列出困难的方法是没有意义的！

与其使用上述方法，不如先为志留纪关联一些线索。例如，骨瘦如柴的鱼的样子。这样，我们可以从一个完形填空开始。比如：“骨鱼出现在[……]纪（4.43-4.19亿年前）”。即使答案和最初的完形填空一样，这两个问题在语义上也不是等价的。针对最初的知识点，你需要生成更多的完形填空。在处理内容相似的序列时，应该始终以递增的方式进行记忆，并且为每一部分都添加一些记忆线索。稍后，您可以针对日期、时代和其他细节生成完形填空。一直以来，你都应该尝试添加新的有趣的线索与学习材料并行记忆，以引入需要长期记忆的有意义的信息。

学习

●使用Ctrl+W（Toolkit >> Calendar）查看复习的日历。双击某一天以查看当天将复习哪些元素。您还可以检查在过去的几天中复习过哪些元素（从Workload 模式切换到 Repetitions 模式）。

●我们可以通过固定在屏幕底部显示的状态栏来查看今天未完成的元素的数量，也可以通过统计窗口（F5）查看。统计窗口也可以保存在默认布局中（使用Ctrl+Shift+F5）。

●阅读（摘录卡片）和学习（问答卡片）的最佳时间分配取决于许多因素：科目和文章的重要性、它们的难度、乐趣因素、个人兴趣、偏好、知识、情绪、生理周期等等。最佳时间分配可以从几秒到几小时不等！这是渐进阅读能力的来源之一。对于一些文本，我们可能会发现很难在几分钟内达到合理的注意力水平。通常，我们只需保留一个句子或段落的最大处理能力即可。在其他非常有趣、写得很好、非常有用或非常重要的文本中，我们的好奇心和与作者矛盾的观点可能会刺激我们一连几个小时不停地继续阅读下去。在渐进阅读中，时间分配的主要标准是我们的注意力集中程度。我们可以一次读上一百篇文章，同时保持高度的注意力和警觉。有些文章会被深入处理，有些则会很快被推迟。专注程度受很多方面影响。它包括上面列出的所有因素：一篇文章的难度可能会影响你的注意力，疲劳会导致我们略过困难的部分，增加对有趣或愉快文本（那些帮助你“度过”糟糕学习日的部分）的分配。

●我们可以使一些低优先级的材料保持“被动复习”的形式（即不生成完形填空）。当然，这些材料会逐渐变得难以回忆或完全忘记。使用“完形填空”的最佳时机是当你注意到材料变得难以回忆时。不要一下子把整篇文章都变成完形填空（除非它很重要）。选择最重要的关键字，只创建一个填空。下一次复习该文章时，您将能够确定哪些关键字必须与完形填空一起使用，以防止忘记关键信息。很难预测你需要生成多少个完形填空来达到对整段文章的完美回忆。有时候，一个完形填空就足够了。但是也有时候，一个段落可能需要一大串完形填空！

●你做得越好，你就越想采用渐进学习。你掌握渐进学习技巧越熟练，采用渐进学习的效率越高，学习同样的知识所需时间越短。对于一个熟练的用户来说，即使是第二天的作业也可以使用渐进学习工具来完成。不过，对于一个初学者来说，可能需要几个月的练习才能真正理解渐进阅读（以及你的记忆）中知识的输入输出。仅此一项就可能使学习一两个月后的考试变得没有效率。

●自动延迟通过减少由于数目巨大而无法掌握的低优先级材料的任务量，使您更接近理想的间隔重复学习。从某种意义上说，自动延迟将高优先级材料（间隔重复）和低优先级材料（传统学习）分开。如果不这样的话，学习节奏会被完全打乱。

●您可以通过添加排序条件中的Randomization来增加材料的随机化程度（在主菜单中：Learn >> Sorting >>Sorting criteria中）。可以分别为摘录卡片和问答卡片设置随机化。它可以使学习材料不带优先级序列的进行复习。

●可以缩短或增加单个元素的复习间隔。如果要在选定的日期安排一篇文章进行复习，可以通过“元素”菜单上的“ Learning >> Reschedule ”（或者使用Ctrl+J）。您也可以使用Learning >> Execute repetition（或者使用Ctrl+Shift+R）。Execute repetition 的功能和 Reschedule类似，不同之处在于重新安排之前将执行一次复习。我们可以根据是否在当天复习过某元素来确定应当使用Execute repetition 还是 Reschedule。Execute repetition需要提供一个分数（除非你在已经在当天完成复习）。

●对于重要性较低的问答卡片，减少优先级（Alt+P）、增加间隔（Ctrl+Shift+R），甚至增加遗忘指数。遗忘指数可以用来保持知识获取率和知识保持率之间的平衡。在超负荷的收藏中给问答卡片赋予低优先级等同于给它设置一个更高的遗忘指数。

●超负荷的学习材料造成的材料丢失程度可在Toolkit >> Statistics >> Analysis >> Use >> Priority protection中查看。一方面你想使Priority protection尽可能的高。另一方面，根据学习进度限制新文章的导入速度可能会使您的收藏缺乏新鲜的元素，从而导致学习的乐趣和动力降低。有些旧的文章可能会因为过载而被推到较低的优先级，但在以后由于不够重要、不够好或过时而被删除。

●我们应当时常考虑我们记忆中的东西的价值。见下方：问答卡片重新评估

●在浏览器中使用Learning >> Spread以及时更改学习材料的分配方案。通过压缩选定时间段内的学习计划，可以加快考试前的学习速度。我们也可以在一次枯燥的考试后重新分配学习材料（用于渐进复习、重新学习、重新设置优先级和/或淘汰旧卡片）。对于后一项工作，我们可以选择每天提供的材料的特定部分。另请参见：Mercy*

●推理/数学中的推导步骤。我们在学习数学时可能会考虑，是否有必要记住数学证明或结论的推导步骤，还是应该只注重结果。这完全取决于我们的目的。如果我们只需要最后的公式，那么花在学习推导步骤上的时间可以花在学习材料的其他重要部分上。如果当下并不确定将来需要什么，我们可以把整个推导过程输入一个摘录卡片，然后只记住最后的公式。在稍后的渐进阅读中，我们将逐步确定推导部分在我们的工作或进一步学习中是否重要。这一块知识将在学习过程中被我们不断地同其他知识点进行比较，从长远来看我们是否要记住细节、是否需要主动回忆、需不需要删除一些步骤、整个推导过程是否多余（或学习成本太高）。当然，推导通常会提高我们有效使用公式的能力。因此，这个决定决不容易。一旦我们添加了推导步骤，我们就可以始终按照它们的优先级来决定我们是否有可能对它们进行充分的复习，从而对我们的知识产生影响。

●你可以把阅读（摘录卡片）和复习（问答卡片）分开。然而，变化是生活的调味品。阅读和复习的随机组合是克服早期版本SuperMemo单调性的一个非常强大的工具。与传统学习或经典的SuperMemo相比，将摘录卡片与问答卡片穿插在一起提供了渐进阅读的许多优势。

○要仅查看摘录卡片（阅读），请选择（1）View >> Outstanding，（2）Child >> Topics，然后选择（3）Process browser> >> Learning >> Learn (Ctrl+L)。

○若要仅复习（问答卡片），请使用类似的方法。

○不过在阅读阶段把问答卡片和摘录卡片混合起来，然后在学习末尾只做问答卡片复习来巩固知识，可能是一个更好的策略。

○最后，坚持优先顺序、自动排序和自动延迟将是最好的最不偏袒的长期策略

●乐趣是成功的关键：如果我们的学习文本太“枯燥”，没有太多意义，太冗长等，学习的乐趣就会下降。如果学习不愉快，它就不太可能有效。如果不喜欢一篇特定的文章，也许维基百科的替代品会更有趣，更有意义？即使这个过程有点长，我们也可以用渐进阅读快速地处理它，用图片来说明，并享受这个过程。

●培养你对知识的渴望：我们必须在知识和它给生活带来的价值之间找到平衡。随着受教育程度的提高，人们对知识的渴求也越来越强烈（知道的越多，不知道的越多）。因此，有一个很好的方法来解决缺乏动力的问题：多学习，看看它会如何影响你和他人的生活。

●渐进阅读的学习速度由我们自己决定！我们可以决定摘录卡片出现的频率（例如，使用A-Factors、优先级、Mercy等）。我们可以确定摘录卡片的保留级别（例如，使用遗忘指数、优先级、自动延迟等）。也可以对任何材料执行强制提前复习（请参见：筛选组复习*）

●我们不要去记忆我们根本不懂的材料！有人希望通过在其他领域做更多的学习有朝一日可以顿悟。不过，我们更可能会对那些无关紧要的东西产生挫败感。如果我们不理解一个术语或概念，我们需要深入探究原因：

○这个词是术语吗？这很容易调查和解决。

○还是材料本身有问题？

○或许我们可以在网上找到另一个选择？

○也许我们可以在网上找到一张很好的图片来说明这个问题？

显然，我们探究一个原因都会额外浪费时间，况且这么做的后果是：在天书般的教科书中，我们只能掌握10%-20%的知识点。不过即使你考试不及格，那10%将来也会有用（如果需要补考的话）。一般来说，学校的任务超过学生所能掌握的限度，这会导致很多压力和挫折感。通过选择SuperMemo，你已经迈出了第一步。现在你需要规律的进行这个过程、用心打造你的长期战略。理解是成功的关键！

●如果想给一个问答卡片打分为Null或Bad，分别按0或1。

●SuperMemo还没有配备工具来帮助您有效地运用知识。它会提高我们的知识储备量，但是。。。必须要注意：不要为了知识而浪费时间获取知识！一定要考虑学习的适用性！幸运的是，随着我们知识的增长，我们有效利用知识的能力也在增长。

重新评估问答卡片

我们应该记住，在的学习过程中添加的所有问答卡片都需要对它们的适用性、公式、重要性、逻辑等给予无休止的关注。在一个计划周密的学习过程中，没必要在复习之余花额外的时间审核问答卡片。但是，当复习到一张问答卡片时，我们应该对以下内容做出快速而近乎本能的评估：

1我真的需要这个东西吗？

2在我所有（想要的）知识中，这个问答卡片最重要的是什么？

3这个问答题难记吗？如果是，为什么？

4这是客观上正确的吗？

5这个问题是不是还可以精简？

6我现在真的需要知道答案吗？

7我需要补充知识来理解这个问答题的所有含义吗？

以下是根据上述问题的答案，我们将采取的一些常规操作：

1编辑卡片。我们将使用Q、A或E等键输入对应字段并对其进行编辑。在更复杂的问答卡片中，我们将使用Ctrl+T在组件之间循环选中，选择一个组件，使用Alt+单击在编辑和拖动模式之间切换，或使用Ctrl+E进入编辑模式

2取消卡片的优先级。对于不够重要或我们不确定是否足够重要的问答卡片，请使用Alt+P降低它们的优先级。我们也可以使用Ctrl+Shift+↓来进行轻微的降级

3重新安排该卡片的复习间隔。如果我们很清楚该卡片，或者出于某种原因希望手动增加（或减少）重复间隔的长度，请按Ctrl+Shift+R选择下一次复习的日期

4搁置当前问答卡片。如果我们确定以后不太可能需要该问答卡片，但希望将其保存在收藏中以供参考或存档，请按Ctrl+D。被搁置的问答卡片将从学习过程中删除

5删除问答卡片。Del键在清除收藏中的垃圾时非常有用，垃圾是由于我们希望知道的信息超出了记忆限度。在编辑模式或拼写卡片中（即Del键实现文本编辑功能时），可能需要使用Ctrl+Shift+Del。请注意，删除SuperMemo中的元素将删除其所有子元素！因此，我们可能希望学习始终使用更为安全的Done选项（Ctrl+Shift+Enter）来代替Del。这将仅删除选中的元素，而保留子元素。

6延迟或忘记问答卡片。如果我们认为这个问答卡片目前太难了，我们可以推迟学习。为此，请选择Ctrl+J以设置新的间隔，或使用Forget将问答卡片添加到挂起队列的末尾。这将给我们一些时间导入一些辅助材料，帮助我们了解该问答卡片。

规划

●使用最小信息原则，即主动回忆中，制定简单的元素比复杂元素带来更好的学习效果。即使一个复杂的元素可能等同于大量简单的元素，简单的元素还是更有效。见：最小信息原则*。

●有些信息是列表形式的。我们应避免这种情况。然而，有些是不可避免的（如神经列表、支流列表、欧盟成员国列表等）。如果需要记忆列表，则应该使用记忆技巧，并试图模拟现实生活中的情况，以降低记忆干扰。另请参见：学习列表

●在SuperMemo中问问题的方式可能与生活中问问题的方式不同。换句话说，我们在SuperMemo中存储的一些材料，现实生活中的情况会使我们无法回忆。换言之，我们需要恰当地描述材料，以便在所有潜在的上下文中最大限度地回忆它。

●记住普遍性的记忆规则。例如，学习一个普遍的数学公式比仅仅学习其使用的例子要好。例子可以用来强调在不同环境中的适用性。

●可以使用Parse HTML（Ctrl+Shift+1）将选定的HTML代码转换为格式（例如，尝试插入<hr>或<br>并使用Parse HTML对其进行解析）。还可以使用此选项删除格式（比如要删除换行符）

●我们可以使用HTML编辑器编辑更复杂的文本。有关详细信息，请参见：在默认HTML编辑器中打开HTML文件*

●在渐进阅读中使用背景颜色样式，以保留文档中使用的原始字体。但是，若要执行此操作，必须取消选中Internet Explorer中的以下选项：工具 >> Internet选项 >> 常规 >> 辅助功能 >> 格式 >> 忽略网页上指定的颜色（Tools >> Internet options >> General >> Accessibility >> Formatting >> Ignore colors specified on webpages）

要了解更多关于高效规划方案的知识，请阅读：有效的学习：处理知识的20条规则

我们的使用感受

在渐进学习中，我们将很快发现为什么自己对学习过程的一些想法可能不是最佳的。以下是我们将在前2-3个月的紧张渐进学习中发现的一些事情：

1认识对考试有益，但从长远来看，回忆对我们的专业技能至关重要

2手工安排复习的时间并不是最适合记忆的；它实际上与SuperMemo的想法完全相反，SuperMemo的意思是在有助于我们稳定记忆的时刻复习学习材料

3手工安排复习顺序并不是最适合我们记忆的（在应对考试方面，筛选组复习是SuperMemo中非常有用的工具）

4对于初学者来说，由于学习曲线陡峭，传统学习可能在很短的时间内（可能长达1-2个月）优于SuperMemo。我们需要先学习渐进阅读的工具集，然后才能从中获益（除非我们采用简单的问答学习，这样的话即使在一周的时间内，我们就能通过SuperMemo达到更好的学习效果）

5我们可以在1-2周内达到95%的知识保留率，条件是我们不推迟复习。然而，如果我们一次将1000页的材料添加到这个过程中，我们将无法按照SuperMemo的计划查看所有的材料，并且我们的知识保留率可能徘徊在60-80%左右，这取决于我们在复习上投入了多少时间

6一旦SuperMemo对我们的记忆和习惯有了一点了解（1-3周），我们将在第一次复习的时候保留大约95%的回忆率（前提是不拖延，并且掌握了处理知识的规则）

7我们会很快发现，在一个段落中挖去多个关键词生成完形填空不是一个好主意（我们可以通过与分隔了相同完形填空关键字的问答卡片的遗忘指数进行比较，或者只是看看这样获得的知识在实践中是如何工作的）

8我们还可以在SuperMemo中查看学习参数，看看不同的学习方法如何影响我们的进步
